用户必须严格遵守中华人民共和国现行的所有网络相关法律、政策及管理条例。这包括但不限于：

2. 《中华人民共和国数据安全法》
3. 《网络信息内容生态治理规定》

## 第二章 内容安全与分级限制
### 第二条 禁止公开传播成人/限制级内容
严禁在公开时间线（Home/Local/Federated）发布被归类为 R18（成人）、包含过度裸露、强烈性暗示或极度令人不适的猎奇内容。如若发现，相关帖子将被立即删除，本站有权立即永久封禁（Global Mute）相关账户，封存相关证据报警，事后执行数据删除。

### 第三条 禁止未经许可的R18内容私信发送
严禁通过私信（DM）向未明确表示同意的用户发送 R18、血腥、暴力或令人不适的内容。若站长收到相关举报，查证属实的，视情节严重程度，给予公开严重警告并限制私信功能，永久封禁账号或上交证据报警。

### 第四条 服务器主动清理权
用户知悉并同意，管理员有权不定期对对象存储（Object Storage）中的私信中的涉色，涉黄以及血腥暴猎奇内容进行清理。若您的内容被判定为包含以上元素，服务器有权在任何时候删除。

## 第三章 版权与诚信
### 第五条 禁止盗用与冒充
严禁将非本人创作的摄影作品据为己有，严禁未经授权下载、截取站内图片用于商业转载或声称他人作品为原创（盗图）。若发生以上事件，站长和管理员有权冻结违规内容并公开警告，若多次涉及相关内容，视情节严重性，除以禁用网盘权限至永久封禁账户的处罚。

### 第六条 禁止无标注的转载
严禁在未清晰标注原作者/出处的情况下，搬运站外或站内的非原创内容。若发生版权纠纷，CaptRAW有权支持原作者维护自己的所有合理合法权益。

## 第四章 机器人与自动化
### 第七条 禁止恶意机器人
严禁部署未经声明、干扰正常社区秩序的自动化脚本或恶意机器人（Bot）。这包括但不限于自动刷屏、自动无差别关注/提及、自动抓取私人数据等行为。用于辅助功能、艺术创作且已声明的正常机器人是允许的，但必须遵守社区规则。若发现机器人正在进行舆论攻击或引战，将立即屏蔽该机器人节点，若造成数据滥用或骚扰，连带封禁所属主账号并执行数据删除。

## 第五章 社区秩序
### 第八条 禁止骚扰与垃圾信息
严禁针对特定用户进行持续性骚扰、人身攻击，或发布未经请求的商业广告（SPAM）。若发现相关内容，则本站有权删除违规内容，临时限制账户功能（如禁止发帖7天），公开警告等，屡教不改者永久封禁并执行数据删除。

## 第六章 违规处理总则
### 第九条 处罚执行
对于违反上述任一条款的用户，管理员有权立刻进行处置，站长拥有最终裁决权，来执行以下措施：
1. 公开警告：针对轻微违规或初犯。
2. 内容删除：移除违规帖子或媒体文件。
3. 功能限制：禁止发帖、禁止评论、禁止私信等。
4. 永久封禁：针对严重违规、多次违规或违反法律底线的行为。
5. 报警处理：涉及违法内容（如传播儿童色情、恐怖主义、严重侵犯隐私等），本站将依法移交司法机关处理并配合调查。
